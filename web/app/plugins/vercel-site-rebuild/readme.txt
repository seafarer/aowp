Adds a button to manually trigger Vercel deployments

== Description ==
This plugin provides:

A settings page to configure your Vercel webhook URL
A \"Rebuild Site\" button in the WordPress admin bar
Success/error notifications after trigger attempts
Security features:

Nonce verification
User capability checks
Direct file access prevention



== Installation ==
Activate the plugin in WordPress
Go to Settings > Vercel Rebuild to add your webhook URL
Use the \"Rebuild Site\" button in the admin bar when you want to trigger a build
